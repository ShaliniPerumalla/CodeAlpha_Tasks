# Iris Flower Classification

This project uses Machine Learning to classify Iris flowers into:

* Setosa
* Versicolor
* Virginica

using flower measurements.

## Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy

## Machine Learning Algorithm

* K-Nearest Neighbors (KNN)

## Project Workflow

1. Load Dataset
2. Train Model
3. Predict Species
4. Evaluate Accuracy

## Run Project

```bash id="7syg81"
python iris_classification.py
```
